# Visual AI (Nano Banana)

You have access to Gemini VLM capabilities via `tool-nano-banana` and the
`nano-banana:nano-banana-expert` agent.

**Delegate to `nano-banana:nano-banana-expert` for:**
- Mockup-to-code conversion workflows
- Structured multi-step visual analysis
- Design fidelity comparison (implementation vs original)
- Iterative visual workflows (analyze → generate → compare)

**Use `tool-nano-banana` directly for:**
- Quick one-off image analysis or description
- Simple generate or compare operations

Requires `GOOGLE_API_KEY` environment variable.
